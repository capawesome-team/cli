## Installation

The Capawesome Cloud CLI can be installed globally via npm.

```bash
npm install -g @capawesome/cli
```

## Help

The Capawesome Cloud CLI ships with command documentation that is accessible with the `--help` flag.

```bash
npx capawesome --help
```
